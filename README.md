## Getting started

Here are steps to get started with using **TinyVec**:

1. Download **TinyVec**.

```bash
git clone https://github.com/kdslfjioasdfj/tinyvec
cd tinyvec
```

2. Build **TinyVec** for your platform.

```bash
premake5 ninja
ninja
```

3. See the documentation for **TinyVec**.
   The documentation is at [docs.md](./docs.md).

4. See the example.
   The example is at [examples/example.c](./examples/example.c). It demonstrates basic usage of **TinyVec.**
